Javascript - джаваскрипт

Javascript пишется в отдельном файле с расширением .js
Подключается Javascript в секции <head>
<script src="somefile.js"></script>

переменные
Задаются словом let (старая версия var)
Например 
  let name = 'Dima';
  let age = 12;
Переменные могут хранить значения и менять их.

Функции в js
Состоит из 
 - слова function, 
 - имени функции, 
 - параметров в круглых скобках ()
 - тела функции в фигурных скобках {}
 - слова return в теле функции - возвращаемый результат работы

 Например:
 function sum(a, b) {
    return a + b;
 }

 вызов функции: имя функции и скобочки с параметрами
 Например:
 let result = sum(5, 8);

 Вывод в консоль:
 console.log('Hello Dima'); // - Напечатает в консоли Hello Dima
